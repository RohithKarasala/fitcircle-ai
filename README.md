FitCircle AI is a modern fitness tracking platform designed to help users build strength, lose fat, and stay consistent through data-driven training.

The application enables users to follow structured workout programs, log sets, weights, reps, and RIR, compare performance with previous sessions, and securely synchronize their training history across devices using Supabase Authentication and PostgreSQL.

Built as a production-style full-stack application, FitCircle AI demonstrates modern web development practices including React, Vite, Supabase, Google OAuth, cloud persistence, responsive UI design, and scalable application architecture.

Current Features
• Google Authentication
• Personalized workout logging
• Progressive overload tracking
• Workout history
• Responsive dashboard
• Secure cloud backend
• User-specific data with Row Level Security (RLS)

Planned Features

• Personal record detection
• Nutrition tracking
• Body measurements
